      ðŸŸ¢ The system will be composed of a sensor that integrates the functionality of two sensors: an accelerometer and a gyroscope, allowing the measurement of the position of the aeronautical object, which are the wings, on a three-dimensional XYZ axis, including their acceleration force, expressed in G's magnitude. Once each magnitude is measured, the system should be able to process this data, which will then be sent to the display to be viewed by the user.

      ðŸŸ¢ In order to execute this co-simulation, the applications were programmed using C++, and compiled into a.ino file, which is the format that the board's microcontroller reads. Then was used the I2C protocol that allows communication between the data monitor of the Arduino IDE and the device that begins to capture the sensor's position on the XYZ axes and print them on the monitor.
